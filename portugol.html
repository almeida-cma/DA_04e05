📚 Dicionário - Estruturas de Controle

Consulte sempre que tiver dúvida sobre o significado de uma palavra:

if → Se (condição)
else → Senão (alternativa)
switch → Escolha (múltiplas opções)
case → Caso (opção do switch)
default → Padrão (opção padrão)
break → Parar (interrompe)
for → Para (repetição contada)
while → Enquanto (repetição condicional)
do → Faça (primeiro executa, depois testa)
true → Verdadeiro (condição verdadeira)
false → Falso (condição falsa)
equals() → Igual (comparação de texto)
