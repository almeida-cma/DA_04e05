📖 O que são Estruturas de Controle?

Estruturas de controle são comandos que permitem que um programa tome decisões e repita ações.

🎯 DOIS TIPOS PRINCIPAIS:
  • 🔀 Condicionais: if, else, else if, switch → tomam decisões
  • 🔄 Repetição: for, while, do-while → repetem ações
💡 ANALOGIA:
  • Condicional: "Se estiver chovendo, leve guarda-chuva"
  • Repetição: "Enquanto houver alunos, chame um por um"

🔀 Estruturas Condicionais

1. if (se)

Executa um bloco apenas se a condição for verdadeira.

package temacontrole;

public class ExemploIf {
    public static void main(String[] args) {
        int idade = 18;

        if (idade >= 18) {
            System.out.println("Maior de idade");
        }
    }
}

2. if / else (se / senão)

Executa um bloco se for verdadeiro, e outro se for falso.

package temacontrole;

public class ExemploElse {
    public static void main(String[] args) {
        int idade = 16;

        if (idade >= 18) {
            System.out.println("Maior de idade");
        } else {
            System.out.println("Menor de idade");
        }
    }
}

3. else if (senão se)

Permite múltiplas condições em sequência.

package temacontrole;

public class ExemploElseIf {
    public static void main(String[] args) {
        int nota = 85;

        if (nota >= 70) {
            System.out.println("Aprovado");
        } else if (nota >= 50) {
            System.out.println("Recuperação");
        } else {
            System.out.println("Reprovado");
        }
    }
}

4. switch (escolha)

Útil quando você tem muitas opções para um mesmo valor.

package temacontrole;

public class ExemploSwitch {
    public static void main(String[] args) {
        int dia = 3;

        switch (dia) {
            case 1: System.out.println("Domingo"); break;
            case 2: System.out.println("Segunda"); break;
            case 3: System.out.println("Terça"); break;
            default: System.out.println("Inválido");
        }
    }
}
⚠️ Não esqueça: Use break no final de cada case!

🔄 Estruturas de Repetição

EstruturaQuando usar
forQuando você sabe quantas vezes vai repetir
whileQuando você não sabe quantas vezes vai repetir
do-whileQuando quer executar pelo menos uma vez

1. for (laço contado)

Repete um número exato de vezes.

package temacontrole;

public class ExemploFor {
    public static void main(String[] args) {
        // Conta de 1 até 5
        for (int i = 1; i <= 5; i++) {
            System.out.println("Número: " + i);
        }
    }
}

2. while (laço condicional)

Repete enquanto a condição for verdadeira.

package temacontrole;

public class ExemploWhile {
    public static void main(String[] args) {
        int contador = 0;

        while (contador < 5) {
            System.out.println("Contador: " + contador);
            contador++;
        }
    }
}

3. do-while (laço condicional)

Executa pelo menos uma vez antes de verificar a condição.

package temacontrole;

public class ExemploDoWhile {
    public static void main(String[] args) {
        int numero = 0;

        do {
            System.out.println("Número: " + numero);
            numero++;
        } while (numero < 5);
    }
}
⚠️ CUIDADO: Sempre tenha uma condição de parada para evitar loop infinito!

💻 Exemplos com Scanner

📖 ENTENDENDO O SCANNER:
  • scanner.nextInt() → Lê um número inteiro (ex: 10, -5, 0)
  • scanner.nextDouble() → Lê um número decimal (ex: 7.5, 3.14)
  • scanner.nextLine() → Lê uma linha de texto (ex: nome completo)
  • scanner.next() → Lê uma palavra (ex: "João")
⚠️ ATENÇÃO: Depois de usar nextInt() ou nextDouble(), use scanner.nextLine() para consumir a quebra de linha!

📌 Exemplo 1: if/else com nextInt()

Lê um número e verifica se é positivo ou negativo.

package temacontrole;

import java.util.Scanner;

public class ExemploPositivo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite um número: ");
        int numero = scanner.nextInt();

        if (numero > 0) {
            System.out.println("O número é positivo");
        } else if (numero < 0) {
            System.out.println("O número é negativo");
        } else {
            System.out.println("O número é zero");
        }

        scanner.close();
    }
}

📌 Exemplo 2: switch com nextInt()

Lê um número de 1 a 7 e exibe o dia da semana.

package temacontrole;

import java.util.Scanner;

public class ExemploDiaSemana {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite um número (1 a 7): ");
        int dia = scanner.nextInt();

        switch (dia) {
            case 1: System.out.println("Domingo"); break;
            case 2: System.out.println("Segunda"); break;
            case 3: System.out.println("Terça"); break;
            case 4: System.out.println("Quarta"); break;
            case 5: System.out.println("Quinta"); break;
            case 6: System.out.println("Sexta"); break;
            case 7: System.out.println("Sábado"); break;
            default: System.out.println("Número inválido");
        }

        scanner.close();
    }
}

📌 Exemplo 3: for com nextDouble()

Lê 3 notas e calcula a média.

package temacontrole;

import java.util.Scanner;

public class ExemploMedia {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        double soma = 0;

        for (int i = 1; i <= 3; i++) {
            System.out.print("Nota " + i + ": ");
            soma += scanner.nextDouble();
        }

        double media = soma / 3;
        System.out.println("Média: " + media);

        scanner.close();
    }
}

📌 Exemplo 4: while com nextLine()

Lê nomes até o usuário digitar "sair".

package temacontrole;

import java.util.Scanner;

public class ExemploWhileNome {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String nome;

        System.out.println("Digite nomes (digite 'sair' para parar):");

        while (true) {
            System.out.print("Nome: ");
            nome = scanner.nextLine();

            if (nome.equals("sair")) {
                break;
            }

            System.out.println("Olá, " + nome + "!");
        }

        System.out.println("Programa finalizado.");
        scanner.close();
    }
}

📌 Exemplo 5: do-while com validação

Pede um número entre 1 e 10 até o usuário acertar.

package temacontrole;

import java.util.Scanner;

public class ExemploDoWhile {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int numero;

        do {
            System.out.print("Digite um número entre 1 e 10: ");
            numero = scanner.nextInt();

            if (numero < 1 || numero > 10) {
                System.out.println("Número inválido! Tente novamente.");
            }

        } while (numero < 1 || numero > 10);

        System.out.println("Número válido: " + numero);
        scanner.close();
    }
}
